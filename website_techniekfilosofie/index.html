Wat is Microsoft?

Microsoft is een corporatie die zich bezig houdt met ontwikkelen van computertechnologieën. Microsoft houdt zich bezig met soft- en hardware. Microsoft is opgericht door Bill Gates en Paul Allen In 1975. Het bedrijf is vooral bekend geworden met het populaire besturingssysteem Windows. Ook heeft Microsoft talloze andere producten en programma’s zoals de office producten, de Xbox gameconsole, de Virtual Reality Bril Hololens, de webbrowser Microsoft Edge, etc..

Geschiedenis

BASIC-Interpreter voor de Altair 8800

MS-DOS is Microsoft's eerste product in hun bestaan, toen nog als Micro-Soft. MS-DOS is een basic interpreter voor de altair 8800, een van de eerste computers zoals we die nu kennen. "Een interpreter is een computerprogramma dat steeds een statement van de broncode van een computerprogramma vertaalt naar een voor de processor begrijpelijke vorm." uitleg interpreter van wikipedia.org.


Windows en Office

Windows is in 1985 uitgebracht als grafische extensie voor MS-DOS. In 1990 bracht Microsoft Office uit, hiermee maakte Microsoft het makkelijker om documenten te maken. Dit is uiteindelijk uitgegroeid tot de microsoft office zoals we die nu kennen. Ook is Windows uitgegroeid tot een van de meest gebruikte besturingssystemen ter wereld.

Windows Besturingssysteem

Windows is een besturingssysteem voor de computer. De eerste versie van dit besturingssysteem is Windows XP, inmiddels is er windows 11. In totaal heeft microsoft 14 aantal versies van haar besturingsysteem uitgebracht voor de pc.

Microsoft Edge

Microsoft Edge is een webbrowsers en is opvolger van het oudere internet explorer.

Xbox

Op het gebied van gaming houdt microsoft zich vooral bezig met de door Microsoft gecreëerde Xbox gameconsole. De Xbox is een van de populairste gameconsoles ter wereld.
